- **Temporal**: Breaking → Evergreen
- **Complexity**: Simple → Deep Technical

- **Social**: Private Save → Public Share

## THE 10 CATEGORIES

### 1. PAPER DECODER
**"Research papers explained from first principles"**

- Position: Evergreen × Deep × Curiosity × Private Save
- Arc: REVELATORY (surface result → deep mechanism → implications)
- Save trigger: Reference utility ("I need to understand this paper")

**Material Cause:**
- Inputs: paper title, key figure/chart from paper, author photos (CC)
- Data: arxiv citations, specific metrics, equations simplified
- Hook image: Author portrait (editorial) + paper figure as holographic panel (Scenario 1 proven)

**Formal Cause (slide sequence, 9 slides):**
| # | Type | Content | Density | Notes |
|---|------|---------|---------|-------|
| 1 | hook | "This paper changes everything about [X]" + author face + paper screenshot | H | Pattern: news-editorial (compose_news_hook.py) |
| 2 | body | "The Problem" — what existing approaches get wrong | M | Context setting |
| 3 | diagram | Key insight as visual flowchart (TikZ) | H | Y5: dual coding — THE core concept visualized |
| 4 | body | "How It Works" — the mechanism in plain language | M | ≤4 chunks per Y1 |
| 5 | body | "The Results" — specific numbers, benchmarks | H | Y4: Authority (cite exact figures) |
| 6 | diagram | Before/after comparison or architecture diagram | H | Y2: PATTERN INTERRUPT (visual shift) |
| 7 | body | "Why It Matters" — real-world implications | M | Bridge to viewer's life |
| 8 | synthesis | 3 key takeaways (numbered, save-worthy) | H | Y3: Serial position — this is remembered |
| 9 | cta | "Follow for daily paper breakdowns" | M | Y4: Reciprocity — value given, now ask |

- Density wave: H-M-H-M-H-H-M-H-M ✓ (never 3H in row)
- Pattern interrupt: Slide 6 (second diagram breaks body monotony)
- Von Restorff: Slide 3 (first diagram — visually distinct from surrounding text)
- Cialdini primary: **Authority** (citing specific papers, researchers, metrics)

**Efficient Cause:**
- Theme: `dark` (academic gravitas, tech credibility)
- Typography: Clean editorial (not all-caps — this is intellectual, not urgent)
- Color: Deep navy + cyan accents (innovation, trust per A6)
- Dual coding: equations → simplified diagrams, data tables → bar charts
- Hook scroll-stop: Real author face + floating paper screenshot (proven 9.5/10)

**Final Cause:**
- Viewer action: SAVE (reference for later reading/sharing in team)
- 24h memory: "That paper about [X] — the key insight is [Y]"
- Return trigger: Series format ("Paper #47") builds collection behavior

**Generalized Prompt Template:**
```
TOPIC: [paper title + arxiv ID]
HOOK: compose_news_hook.py with author portrait + floating paper figure
  Image prompt: "editorial portrait of [author], face preserved +
  paper figure as holographic panel, dark background, blue accent"
SLIDES: problem → mechanism diagram → plain explanation → results →
  architecture diagram → implications → 3 takeaways → CTA
TONE: Intellectual curiosity, not hype. "Here's what they found" not "INSANE!"
```

## UNIVERSAL RULES (apply to ALL categories)

1. **≤4 chunks per slide** (Y1). If you need 5+ points, split into 2 slides.
2. **Pattern interrupt every 2-3 slides** (Y2). Use: diagram, comparison, color shift, or layout change.
3. **Density wave** (I3). Never 3 high-density slides consecutively. Insert M or L breathers.
4. **First slide = amygdala trigger** (P2). Each category has its own trigger type.
5. **Synthesis = THE save slide** (Y3 + P4). This is what they screenshot. Make it a standalone reference card.
6. **No-text seal on all AI images** (T6/A7). Always end image prompts with the seal.
7. **Dual code the hardest concept** (Y5). The single most complex idea → diagram or visual.
8. **CTA matches save trigger** (P4). Utility categories → "Save this". Social categories → "Share/Comment".
